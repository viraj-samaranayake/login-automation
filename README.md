# Login Automation - Selenium WebDriver

This project is a Selenium WebDriver-based automation framework developed to validate the login functionality of a web application.  
It demonstrates best practices in UI test automation, including maintainable design patterns, configurable execution, and scalable test structure.

Repository: https://github.com/viraj-samaranayake/login-automation

---

## üìå Project Objective

The main objective of this project is to automate and validate user login scenarios such as:
- Valid login testing

- Basic authentication flow verification

This framework can be easily extended to cover additional UI test cases.
